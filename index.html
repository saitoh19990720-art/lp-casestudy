aqua festa 2026

推し活カルチャーイベント ／ 世界観設計

核：「推しを“空気ごと”持ち込める場所。」騒がしくなりがちな推し活LPに、水と透明感の静けさを持ち込み、情報設計で“好き”を整理した。

選択理由

淡色グラデ＋静かなヒーロー（「空気をもう一度。」）。量産型フェスLPと真逆の方向で記憶に残す。

棄却した案

原色・特大キャッチ・CTA連打のギラギラ全部盛りフェスLP。

賭けた点

イベントLPなのに“静けさ”を主役に。賑やかさより余韻で惹く判断。

構成：hero → コンセプト → 出演 → 見どころ → チケット・特典 → 会場 → 応募

Lumière｜銀座プライベートサロン

高級美容サロン ／ 商用ランディングページ

核：「静かな個室で、ただあなたのために。」余白と細い線で“品”を出し、カウンセリング起点の導線に整えた商用LP。

選択理由

高コントラストを避け、余白・階層・明朝寄りの上品さで信頼感を作る。

棄却した案

大きいCTAを連打し割引を煽る量産型美容LP（安く見える）。

賭けた点

主CTAを「初回30分無料カウンセリング」の1点に絞る情報設計の自制。

構成：hero → コンセプト → 担当セラピスト → メニュー → お客様の声 → ご来店の流れ → 予約

似合いどころ診断

インタラクティブ診断ツール ／ ロジック設計

核：感覚的な「似合う」を、シーズン・サブタイプ・骨格・顔タイプの多軸で構造化。静的LPでなく、多ステップの状態遷移をJSで設計した。

選択理由

分岐する設問をstate遷移で管理＝「ロジックも組める」を実証。

棄却した案

当てずっぽうの占い風・結果を断定して煽るタイプ。

賭けた点

結果を「断定」でなく「整理する」フレーミングに。“盛らない”美学。

構成：シーズン → サブタイプ → 2ndシーズン → 骨格 → 顔タイプ → 結果（多分岐）
